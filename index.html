O curso de SCRUM Master é um treinamento, que tem como objetivo capacitar os participantes a atuarem como SCRUM Master em projetos de desenvolvimento de software, utilizando o SCRUM como framework de gestão de projetos.

Este treinamento apresenta o Framework Scrum que se caracteriza como uma abordagem para o planejamento empírico do produto. Durante o treinamento, os participantes irão entender como um simples - mas eficiente - processo como Scrum pode transformar os resultados de uma empresa, permitindo entregas a cada iteração, com agilidade necessária para responder rapidamente a requisitos que sofram mudanças constantes.
